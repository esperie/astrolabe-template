description: Save progress and close the session cleanly — final vet, update session notes, codify learnings. Never wrap up with failing tests or an unconverged step.
---

Wrap up the session:

1. **Final `/vet`** — tests green, every claim sourced, no canon contradiction, no unconverged finding. **Do NOT wrap up otherwise** — fix first (`rules/redteam-mandatory.md`).
2. **Session notes** — update `.session-notes` (or the workspace note): what changed, what's **verified** (with the convergence verdict), what's **open/next**, and the entry point to resume.
3. **Codify** — capture durable learnings or new instincts worth carrying forward (note them for `/codify`; `rules/learned-instincts.md` is auto-generated by the session-end hook).
4. **Summary** — what was delivered, the verification state, and the next step.

Leave the repo clean and resumable.
