# No Stubs or Placeholders

## Scope
All documents in this repository.

## MUST NOT Rules

### 1. No Placeholder Content
Documents MUST NOT contain:
- TODO markers without implementation
- "[INSERT HERE]" placeholders
- Empty sections
- "TBD" or "Coming soon" markers

### 2. Create, Don't Note
When a gap is discovered, fill it with real content or flag it to the user. Do not leave stubs.
