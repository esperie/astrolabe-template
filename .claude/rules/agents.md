# Agent Orchestration Rules

## MUST Rules

### 1. Review After Significant Changes
After significant document changes, delegate to **intermediate-reviewer** for accuracy/consistency review.

### 2. Security Review Before Commits
Before committing changes involving personal details or client information, delegate to **security-reviewer**.

### 3. Knowledge Base First
Before generating any CV, bio, pitch, or content, ALWAYS read the relevant `docs/` files first. Never generate from memory alone.

### 4. Specialist for Specialist Work
- CV/bio generation → **branding-strategist**
- Content planning → **content-strategist**
- Standards questions → appropriate standards expert
